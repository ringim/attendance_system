## Query Routing

### Without search_path

```sql
-- Must specify schema explicitly
SELECT * FROM attendance_system.employees;
SELECT * FROM inventory_app.products;
SELECT * FROM crm_platform.customers;
```

### With search_path (automatic)

```sql
-- Set once per connection
SET search_path TO attendance_system;

-- Now queries use the schema automatically
SELECT * FROM employees;  -- attendance_system.employees
SELECT * FROM devices;    -- attendance_system.devices
```

